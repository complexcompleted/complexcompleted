HTML Table Cheat Sheet

Table Tags

Tag

Name

Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows that define the column headers of the table.
<tbody> Table Body The set of rows containing the actual table data.
<tr> Table Row The table row container.
<td> Table Data (Column) The data container for that column and row.
<tfoot> Table Foot The set of rows defining the footer of the tabld (not used often).

Table Attributes

Attribute

Name

Description
<colspan> Column Span Defines how many elements a data element <td> should span.
<rowspan> Row Span Defines how many elements a data element <td> should span.
